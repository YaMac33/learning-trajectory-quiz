# LT Quiz（Learning Trajectory Quiz）

ブラウザのみで動作する、静的なクイズ学習Webアプリです。  
GitHub Pagesなどの静的ホスティング環境でそのまま公開できます。

---

## 📌 概要

LT Quiz は以下の特徴を持つシングルHTML構成のクイズアプリです。

- 問題一覧表示
- カテゴリ・出典別グループ表示
- キーワード検索
- 未回答／不正解／お気に入りフィルタ
- 問題回答＋即時正誤判定
- 詳細解説の展開表示
- ランダム演習モード
- 学習ダッシュボード（正答率・進捗表示）
- ローカルストレージ保存（履歴・お気に入り）

サーバーサイド不要。完全クライアントサイド動作です。

---

## 🗂 ファイル構成

最低構成：

```
/index.html        ← 本ファイル
/questions.json    ← 問題データ（任意）
```

※ `questions.json` が存在しない場合は、内部のモックデータ（FALLBACK_QUESTIONS）を使用します。

---

## 📄 questions.json 仕様

配列形式で問題を定義します。

```json
[
  {
    "id": "ap-tech-vcs-distributed",
    "source": "令和6年秋期　問19",
    "category": "テクノロジ系 / ソフトウェア",
    "question": "問題文...",
    "choices": [
      {
        "text": "選択肢A",
        "overview": "誤り。〜",
        "detail": "詳細解説...",
        "link": "https://example.com" 
      }
    ],
    "correct_indices": [2]
  }
]
```

### 必須項目

- `id`（一意）
- `question`
- `choices`
- `correct_indices`

### 任意項目

- `source`
- `category`
- `overview`
- `detail`
- `link`

## 🧠 機能詳細

### 1. 問題一覧画面

- 親カテゴリフィルタ
- 出典（開催期）グループ切替
- キーワード検索（ID / 問題文 / 出典）
- 未回答のみ表示
- 不正解のみ表示
- お気に入りのみ表示

---

### 2. クイズ画面

- 複数選択対応（checkbox）
- 回答後に正誤表示
- 正解／不正解ごとの色分け
- 各選択肢の解説表示（トグル式）
- 参考リンク表示

---

### 3. ダッシュボード

表示内容：

- 全体正答率
- 学習進捗率
- カテゴリ別正答率
- ランダム演習モード

#### ランダム演習モード

- 未回答から10問
- 不正解問題から10問
- お気に入りから10問

---

## 💾 データ保存

以下はブラウザの LocalStorage に保存されます。

- 回答履歴
- 最新正誤結果
- お気に入り情報

キー：

```
ltQuizLogs_v2
ltQuizBookmarks
```

サーバー送信は一切行いません。

---

## 🚀 公開方法（GitHub Pages）

1. リポジトリ作成
2. `index.html` を配置
3. Settings → Pages → Deploy from branch
4. main / root を指定

以上で公開可能です。

---

## ⚙ カスタマイズ

### カラー変更

`:root` の CSS変数を編集してください。

例：

```
--c-primary
--c-success
--c-danger
```

---

### 問題追加

`questions.json` に問題を追記するだけで自動反映されます。

---

## 🔐 セキュリティについて

- 完全クライアントサイド動作
- サーバー通信なし
- ローカル保存のみ

※ そのため、認証機能・ユーザー管理機能はありません。
